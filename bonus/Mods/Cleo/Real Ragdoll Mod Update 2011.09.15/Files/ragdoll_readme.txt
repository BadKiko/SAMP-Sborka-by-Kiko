Ragdoll mod for GTA:SA
 by Ryosuke839 2011.04.12
 http://hotmist.ddo.jp/plugin/ragdoll/index.html


Installation
 Install ASI loader.
 Put every files in this zip archive to GTA San Andreas directory.


Notice
 This mod requires strong cpus.
 We support gta_sa.exe with 14,386,176 bytes (EU1.0) and 14,383,616 bytes (US1.0).
