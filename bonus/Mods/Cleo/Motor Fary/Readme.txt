Вставляйте файл engine.cs в папку CLEO.

Управление: нажимаете клавишу "." (английская раскладка клавиатуры), чтобы завести/заглушить мотор. Если при движении автомобиля заглушить мотор, то машина будет катиться по инерции какое-то время.
Нажимаете клавишу "L" для включения/выключения фар.
Нажимаете клавишу "O" (английская раскладка клавиатуры) для блокировки дверей.
