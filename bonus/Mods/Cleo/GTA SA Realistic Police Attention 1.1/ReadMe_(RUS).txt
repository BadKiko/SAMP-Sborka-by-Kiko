San Andreas - Realistic Police Attention
Версия: 1.1
Автор: gtafan222
--------------------------------------------------------------------------------------------------------------------------------------------
Описание: Данный CLEO скрипт улучшает полицейское внимание т.е. копы сильнее обращают внимание на преступления, совершенные CJ'еем; Увеличился набор звезд за любое правонарушение; "Поведение" жильцов при поиске копами CJ'я тоже изменилось - Всё по-обычному, если розыск игрока не достиг 4 звезд (0-3). Число людей на улице убавится наполовину, но копов уже будет больше, если у вас 4 звезды. Если розыск больше 4 звезд (5-6), то число копов увеличится в 2 раза, в отличии от обычной Сани, на улчце не будет пешеходов. Также с 4 звезд отключается движение поездов по Сан Андреас'у. В версии 1.1 это всё можно настроить.
-------------
В данном архиве есть 2 варианта этого скрипта - Для CLEO 3 и для CLEO 4. Отличия в том, что скрипт для CLEO 4 может настраиваться через INI файл, а CLEO 3 имеет готовые настройки, которые можно изменить только через Sanny Builder.

Установка версии для CLEO 3: Убедитесь в том, что на вашу San Andreas установлен CLEO 3. Далее просто скопируйте файл "RPM_PoliceAttention.CS" в папку CLEO.
Установка версии для CLEO 4: Убедитесь в том, что на вашу San Andreas установлен CLEO 4. Далее просто скопируйте файлы "RPM_PoliceAttention.CS" и "RPM.INI" в папку CLEO. Перед игрой, файл INI можно настроить (см. ниже).
-------------
Конфигурационный файл RPM.INI для скрипта CLEO 4:

[POLICE_ATTENTION]
SAttentionDensity = (> 0.0) Задает внимание полиции, если игрок носит обычную одежду. Чем больше значение, тем больше внимание полиции. Наименьшее значение (0.0) дает полное игнорирование полиции. Слишком большие значения (> 10.0), могут привести к вылету из игры
SMaxWantedLevel = (0 - 6) Задает максимальный уровень розыска, которого можнт достичь игрок, если ности обычную одежду. Если поставить 0, то менты вас НИКОГДА не тронут.
SMilitaryZoneWL = (0 - ВЫКЛ/1 - ВКЛ) Задает розыск (5 звезд) при попадании на военную базу. Если отключить, то в этом случае, при нахождении вас на военной базе, копы искать Карла не будут. При этом он должен носить обычную одежду.
CAttentionDensity = (> 0.0) Задает внимание полиции, если игрок носит ПОЛИЦЕЙСКУЮ одежду. Чем больше значение, тем больше внимание полиции. Наименьшее значение (0.0) дает полное игнорирование полиции. Слишком большие значения (> 10.0), могут привести к вылету из игры 
CMaxWantedLevel = (0 - 6) Задает максимальный уровень розыска, которого можнт достичь игрок, если ности ПОЛИЦЕЙСКУЮ одежду. Если поставить 0, то менты вас НИКОГДА не тронут.
CMilitaryZoneWL = (0 - ВЫКЛ/1 - ВКЛ) Задает розыск (5 звезд) при попадании на военную базу. Если отключить, то в этом случае, при нахождении вас на военной базе, копы искать Карла не будут. При этом он должен носить ПОЛИЦЕЙСКУЮ одежду.

[TRAFFIC_BEHAVIOR]
PlayerWLToChange = (2 - 5) Это средний розыск, при котором поведение людей будет меняться. Если розыск больше среднего, то ориентируйтесь на параметры "MaxW...", а если меньше, то на "MinW...". В общем, этот розыск относится к параметрам "MedW..."
MinWCarDensity = (0.0 - 10.0) Частота движения автомобилей при меньшем розыске. Наименьшие значения отключают движение автомобилей и патрульных машин. Слишком большие значения могут привести к вылету из игры.
MinWPedDensity = (0.0 - 10.0) Частота движения пешеходов при меньшем розыске. Наименьшие значения отключают движение пешеходов. Слишком большие значения могут привести к вылету из игры.
MinWTrainTraffic = (0 - ВЫКЛ/1 - ВКЛ) Движение поездов по штату при малом розыске.
MedWCarDensity = (0.0 - 10.0) Частота движения автомобилей при среднем розыске. Наименьшие значения отключают движение автомобилей и патрульных машин. Слишком большие значения могут привести к вылету из игры.
MedWPedDensity = (0.0 - 10.0) Частота движения пешеходов при среднем розыске. Наименьшие значения отключают движение пешеходов. Слишком большие значения могут привести к вылету из игры.
MedWTrainTraffic = (0 - ВЫКЛ/1 - ВКЛ) Движение поездов по штату при среднем розыске.
MaxWCarDensity = (0.0 - 10.0) Частота движения автомобилей при высшем розыске. Наименьшие значения отключают движение автомобилей и патрульных машин. Слишком большие значения могут привести к вылету из игры.
MaxWPedDensity = (0.0 - 10.0) Частота движения пешеходов при высшем розыске. Наименьшие значения отключают движение пешеходов. Слишком большие значения могут привести к вылету из игры.
MaxWTrainTraffic = (0 - ВЫКЛ/1 - ВКЛ) Движение поездов по штату при высшем розыске.
